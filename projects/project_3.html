Analytics Dashboard

Project Overview

As the final section of a course-long project to run a server and host a custom domain, this analytics dashboard was meant to use all the skills we had learned up to this point and incorporate and graphical display of visitors data.

In 2-person team, I held the role of Full-Stack engineer, co-building and designing front-end design in HTML, CSS and JavaScript. The backend was developed on an Apache2 server, Ubunutu and maintained with a MySQL database.

Unfortunately, this site is no longer running as it depended on student credits funding the DigitalOcean subscription that supported the server. However, the GitHub repository is available with the above screenshots. However, at the end of the project, it was capable of recording and storing records of a users static browser, performance and activity (keyboard and mouse) data, and displaying it back the user for further analysis.
