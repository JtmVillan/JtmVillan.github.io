Project Overview

This project is the latest published version of my personal portfolio website. Originally, my first portfolio site was built as course project to learn vanilla HTML, CSS, JavaScript and some responsive design. However, that site was purely for practice (exploring the various tags and capabilities and basid front-end design without relying on external tools) and never published online.

With this new site, I originally adapted a Bootstrap template, called DevFolio, and eventually ended up rewriting most of it keeping some basic structure and libraries used. Many iterations and designs were ran through before deciding on this more simplistic approach.

Throughout my development process I've learned and re-learned about how important response design is and what I am able to learn from other front-end designs that inspired my site. And I plan to conintually update over time.
